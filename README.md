# 💜 Purplle Clone

A front-end clone of the Purplle website created using HTML and CSS. This project replicates the design and layout of the original website while focusing on responsive and user-friendly interface development.

## 🚀 Features
- Responsive Design
- Homepage Layout
- Navigation Bar
- Product Sections
- Attractive UI Design

## 🛠️ Technologies Used
- HTML5
- CSS3

## 🎯 Purpose
This project was built to practice and improve my front-end web development skills by recreating a real-world e-commerce website interface.
